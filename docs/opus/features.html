功能模块

部分动态相关接口请求存在 features 参数, 意为功能
主要用于控制返回结果中的 modules 中的内容, 意为模块
此两者在一定程度上为对应关系, 但也存在大量例外
本页文档整理了大部分常见的已知功能与模块以供参考

请求 URL 参数, 用 , 分隔
